将可疑文件上传至威胁情报平台进行自动化分析，快速判断文件是否为恶意样本。

<Callout title="前置条件" type="info">
  需要[设置威胁情报 API Key](/credential#设置-威胁情报-api-key)。SIREN 目前支持[微步在线云 API](https://x.threatbook.com/v5/apiDocs#/)。
</Callout>

## 上传样本

```bash tab="本地模式"
./siren ti <file path>
```

```bash tab="远程模式"
>>> ti <Client ID> <file path>
```

<Accordions>
  <Accordion title="示例输出">

```bash
[+] Request succeeded. Response:  {
    "sha256": "6e123e7f3202a8c1e9b1f94d8941580a25135382b99e8d3e34fb858bba311348",
    "permalink": "https://s.threatbook.cn/search?query=6e123e7f3202a8c1e9b1f94d8941580a25135382b99e8d3e34fb858bba311348&type=sha256"
}
```

  </Accordion>
</Accordions>

返回字段说明：

| 字段 | 说明 |
| --- | --- |
| `sha256` | 样本的 SHA256 哈希值 |
| `permalink` | 威胁情报平台的分析结果链接，可直接在浏览器中打开查看详细分析报告 |
