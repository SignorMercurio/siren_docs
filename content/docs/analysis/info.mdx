基于进程 PID、监听端口或进程名称，快速获取进程详细信息，包括：

- 进程基本信息（PID、PPID、用户、启动时间、运行时长等）
- 进程命令行参数和可执行文件路径
- 进程打开的文件和网络连接
- 进程关联的容器信息（如有）
- 进程关联的 systemd 服务（如有）

![info](../../img/info.png)

## PID 查询

```bash tab="本地模式"
./siren info <PID>
```

```bash tab="远程模式-SIREN Server"
>>> info <Client ID> <PID>
```

## 监听端口查询

```bash tab="本地模式"
./siren info -p <Port>
```

```bash tab="远程模式-SIREN Server"
>>> info <Client ID> -p <Port>
```

## 名称查询

支持模糊匹配：

```bash tab="本地模式"
./siren info -n <Process Name>
```

```bash tab="远程模式-SIREN Server"
>>> info <Client ID> -n <Process Name>
```
