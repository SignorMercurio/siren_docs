## 客户端管理

远程模式下，每个客户端都有唯一的 ID，可以通过该 ID 对客户端进行管理。

列出所有客户端：

```bash title="SIREN Server"
>>> ls
```

为客户端添加备注信息：

```bash title="SIREN Server"
>>> note <Client ID> <Note>
```

相关备注会在 `ls` 中显示。

## 远程命令执行

通过 tmux 启动新窗口，获取一个**全交互式**的远程 Shell，可以正常使用 vim、top 等交互式命令、方向键、Tab 补全等：

```bash title="SIREN Server"
>>> shell <Client ID>
```

运行单条命令：

```bash title="SIREN Server"
>>> run <Client ID> <Command>
```
