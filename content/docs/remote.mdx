获取一个**全交互式**的远程 Shell，可以正常使用 vim、top 等交互式命令、方向键、Tab 补全等：

```bash title="SIREN Server"
>>> shell <Client ID>
```

运行单条命令：

```bash title="SIREN Server"
>>> run <Client ID> <Command>
```
