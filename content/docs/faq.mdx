<Accordion title="如何升级 SIREN？">

SIREN 客户端与服务端均支持一键版本升级，使用 `-y` 参数可自动确认升级。

客户端升级：

```bash
./siren upgrade
```

服务端升级：

```bash
./siren_server upgrade
```

</Accordion>

<Accordion title="远程模式服务端有哪些快捷键？">

参考 [readline 快捷键](https://github.com/ergochat/readline/blob/master/docs/shortcut.md)。

</Accordion>

<Accordion title="SIREN 与云安全中心 Agent 是否兼容？">

SIREN 与云安全中心 Agent 兼容性良好，但使用过程中可能会触发告警：

1. 使用远程 Shell 功能时，可能触发 `异常的伪终端Shell创建行为` 告警、`反弹Shell` 告警，但不会被拦截。
2. 使用自动化信息收集功能时，可能触发 `蠕虫病毒命令` 告警，但不会被拦截。

</Accordion>

<Accordion title="如何在容器环境中使用 SIREN？">

部署在宿主机上的 SIREN 可以直接收集容器内恶意文件、进程、网络连接等从宿主机层面可观测的项；命令执行历史、计划任务等，则需要通过容器内部启动 SIREN 来收集。

</Accordion>

<Accordion title="启动时报错：Error reading config file">

SIREN 在默认查找路径下未能找到配置文件，详见 [配置文件](/config)。

</Accordion>
