## 标准消息

```go title="internal/protocol/protocol.go"
type Header struct {
	MsgType uint8
	DataLen uint64
}

type Message struct {
	Header Header
	Data   []byte
}
```

即：`1 字节消息类型 | 8 字节消息长度 | 消息内容`。

## Raw 消息

顾名思义，Raw 消息无固定格式，客户端发送消息至服务端后，如果消息的首字节不在标准消息类型标识范围内则被视为 Raw 消息，读取至 `\n` 并直接打印，随后切换回读取标准消息的模式。

主要适用于客户端日志直传场景。


### forward / stopforward

```mermaid
sequenceDiagram
    autonumber
    participant CS as Client-side Service
    participant C as Client
    participant S as Server
    participant L as Local Env
    S->>S: net.Listen()
    loop
        L->>S: listener.Accept()
        S->>C: MsgPortForward
        C->>CS: net.Dial()
        L->>S: [Request]
        S->>C: MsgPortForwardData
        C->>CS: [Request]
        CS->>C: [Response]
        C->>S: MsgPortForwardData
        S->>L: [Response]
    end
    S->>S: listener.Close()
    S->>C: MsgPortForwardStop
    C->>C: conn.Close()
```
