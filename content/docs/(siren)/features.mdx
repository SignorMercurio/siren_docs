## 信息收集

自动化采集受害主机上的关键安全信息，支持按模块独立开关。

<Cards>
    <Card title="基础系统信息" icon={<Info />} href="/recon">主机名、操作系统、内核版本、CPU、内存、磁盘等</Card>
    <Card title="用户" icon={<Users />} href="/recon">用户列表、登录历史、sudoers、影子文件等</Card>
    <Card title="SSH" icon={<Terminal />} href="/recon">SSH 配置、authorized_keys、known_hosts 等</Card>
    <Card title="进程" icon={<Activity />} href="/recon">进程列表、CPU/内存 Top 20、隐藏进程检测</Card>
    <Card title="网络" icon={<Network />} href="/recon">网络连接、监听端口、DNS 配置、ARP 表等</Card>
    <Card title="计划任务" icon={<Clock />} href="/recon">crontab、at 任务、Windows 计划任务等</Card>
    <Card title="服务与启动项" icon={<Play />} href="/recon">systemd 服务、init 脚本、Windows 服务等</Card>
    <Card title="环境变量" icon={<Code />} href="/recon">系统环境变量、LD_PRELOAD 等敏感变量检查</Card>
    <Card title="文件" icon={<FileText />} href="/recon">近期修改文件、大文件、SUID 文件、敏感目录等</Card>
    <Card title="Rootkit" icon={<Cpu />} href="/recon">内核模块检查、LD_PRELOAD 检查、系统完整性验证</Card>
    <Card title="Webshell" icon={<Globe />} href="/recon">Web 目录扫描，基于关键字和正则匹配的 Webshell 检测</Card>
    <Card title="应用" icon={<Package />} href="/recon">已安装软件包、运行中的 Web 应用/中间件</Card>
</Cards>

## Web 控制台

浏览器图形化操控面板，所有远程模式功能无需 tmux 和 REPL 也能使用。

<Cards>
    <Card title="浏览器控制台" icon={<LayoutDashboard />} href="/webui">统一入口访问客户端管理、Shell、Agentic IR、取证文件浏览等功能</Card>
    <Card title="账号鉴权" icon={<Lock />} href="/webui#账号鉴权">SQLite + bcrypt 账号登录，通过 CLI 管理用户</Card>
    <Card title="多标签 Shell" icon={<Terminal />} href="/webui#shells">WebSocket 承载的交互式终端，切换页面会话不中断</Card>
    <Card title="浏览器内 AIR" icon={<Brain />} href="/webui#agentic-ir">无需登录控制主机即可启动 AI Agent 完成应急响应</Card>
    <Card title="浏览器内调查" icon={<Key />} href="/webui#investigations">执行 AccessKey 调查和 Recon，查看真实阶段进度并跳转报告</Card>
    <Card title="取证文件浏览" icon={<FolderTree />} href="/webui#artifacts">Markdown 预览、悬浮目录、二进制识别、文件下载，文件树宽度可拖拽</Card>
    <Card title="SOAR 快捷操作" icon={<MousePointerClick />} href="/webui#clients">allow/deploy/clean 一键调用，带确认与 toast 提示</Card>
