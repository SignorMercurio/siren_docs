SIREN 服务端内置模型上下文协议 (MCP) 服务器，使 AI 助手能够直接与 SIREN 交互，在受害主机上执行命令、获取安全告警详情等。这是 [Agentic 应急响应](/air) 的基础能力。

## 配置

MCP 服务默认监听 `8080` 端口，可通过[配置文件](/config#服务端配置)中的 `mcpPort` 字段修改。

将以下配置添加到 MCP 客户端（如 Claude Desktop、Cursor 等）：

```json title="mcp.json"
{
  "siren": {
    "type": "http",
    "url": "http://<SIREN Server IP>:8080/mcp"
  }
}
```

## 可用工具

| 工具 | 描述 |
| --- | --- |
| `ls` | 列出已连接的客户端 |
| `run` | 在指定客户端上执行命令。参数：`Client ID`、`Command` |
| `get_alarm_detail` | 通过 SOAR 平台获取指定的安全告警详情。参数：`Alibaba Cloud UID`、`Security Center Event ID` |
