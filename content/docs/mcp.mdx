SIREN 服务端包含模型上下文协议 (MCP) 服务器，用于与 AI 助手集成。

## 配置

添加到你的 MCP 客户端配置中（MCP 服务监听端口可配置）：

```json title="mcp.json"
{
  "siren": {
    "type": "http",
    "url": "http://<SIREN Server IP>:8080/mcp"
  }
}
```

## 可用工具

| 工具 | 描述 |
| --- | --- |
| `ls` | 列出已连接的客户端 |
| `run <Client ID> <Command>` | 在指定客户端上执行命令 |
| `get_alarm_detail <Alibaba Cloud UID> <Security Center Event ID>` | 通过 SOAR 平台获取指定的安全告警详情 |
