当发现泄露的云上 AccessKey 时，可以快速调查其调用历史以评估影响范围、梳理止血策略。

<Callout title="前置条件" type="info">
  需要在控制主机上[设置 SOAR 平台访问凭证](/credential#设置-soar-平台访问凭证)（内部环境已设置）。
</Callout>

## 使用方式

调查 AccessKey 调用历史并生成详细报告：

```bash title="SIREN Server"
>>> ak <Alibaba Cloud UID> <AccessKey ID>
```

该命令执行以下操作：
1. 查询 SOAR 平台获取 AccessKey 调用历史
2. 生成包含调查结果的详细 Markdown 报告
3. 如设置了 OSS 访问凭证，自动将报告上传至 OSS（同样支持 MkDocs 展示）
