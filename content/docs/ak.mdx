内部环境下，在控制主机上 [设置 SOAR 平台访问凭证](/credential#设置-soar-平台访问凭证) 后，支持调查 AccessKey 调用历史并生成详细报告：

```bash title="SIREN Server"
>>> ak <Alibaba Cloud UID> <AccessKey ID>
```

该命令执行以下操作：
1. 查询 SOAR 平台获取 AccessKey 调用历史
2. 生成包含调查结果的详细 Markdown 报告
3. 如设置了 OSS 访问凭证，自动将报告上传至 OSS（同样支持 MkDocs 展示）

  <Accordion title="示例">

    ```bash title="SIREN Server"
    >>> ak 1601097845544644 AKIA1234567890123456
    [*] Investigating AccessKey...
    [+] Investigation completed. Report saved to: AK_1601097845544644_AKIA1234567890123456_2024-01-15_14:30:25.md
    ```
