<Callout title="核心原则" type="info">
  SIREN 的基础功能（信息收集、文件上传、进程分析 等）**无需依赖**任何外部基础设施即可使用。以下基础设施均为可选，用于增强特定功能体验。
</Callout>

## 通用基础设施

| 基础设施 | 对应功能 | 必要性 |
| -------- | -------- | -------- |
| OSS Bucket 及访问凭证 | [信息收集报告上传](/recon)、[数据展示](/recon#查看数据)、[文件上传](/analysis/upload) | 可选 |
| 微步在线威胁情报 API Key | [快速样本分析](/analysis/ti) | 可选 |
| 控制主机（需安装 tmux） | 所有远程模式功能 | 远程模式必需 |

## 内部环境额外基础设施

| 基础设施 | 对应功能 |
| -------- | -------- |
| 云存储网关与极速同步组 | [数据展示](/recon#查看数据)（MkDocs 自动同步） |
| MkDocs 站点 | [数据展示](/recon#查看数据)（搜索 + AI 分析） |
| AI 分析 API 服务及相关 Key | [数据展示](/recon#查看数据)（AI Summary 功能） |
| SOAR 平台及访问凭证 | [云助手一键部署](/#内部环境通过云助手一键部署)、[自动放行受害主机](/index#手动部署)、[AccessKey 调查](/ak)、[MCP 工具 `get_alarm_detail`](/mcp#可用工具) |
