SIREN Server 提供了一些快捷操作，方便在服务端控制台中直接执行常用操作。

## 执行 Shell 命令

使用 `!` 前缀在服务端执行 Shell 命令：

```bash title="SIREN Server"
>>> ! <Command>
```

例如查看服务端当前目录：

```bash title="SIREN Server"
>>> ! pwd
```

## 设置环境变量

使用 `env` 直接在服务端设置环境变量，常用于配置各类[访问凭证](/credential)：

```bash title="SIREN Server"
>>> env <KEY>=<VALUE>
```

例如设置 OSS 访问凭证：

```bash title="SIREN Server"
>>> env OSS_ACCESS_KEY_ID=<YOUR_ACCESS_KEY_ID>
>>> env OSS_ACCESS_KEY_SECRET=<YOUR_ACCESS_KEY_SECRET>
```

## 查看帮助

```bash title="SIREN Server"
>>> help
```
